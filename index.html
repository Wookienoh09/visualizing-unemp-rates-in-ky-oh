Description

Map details

Author: Kangwook Noh

This map presents the latest available unemployment rate data for Kentucky and Ohio in 2022, categorized by county. Notably, a distinct pattern emerges as higher unemployment rates are prominently observed in the eastern counties along Appalachia for both states. Conversely, counties situated in the central regions of both Kentucky and Ohio exhibit lower unemployment rates. Additionally, there is a noticeable consistency in unemployment rates among counties along the state border, highlighting a shared economic trend in those areas.

Below map shows the change in unemployment rates over time between 2000 to 2022. Map descriptions on the image.

Data

State and County level data (2022) extracted from the 2022 version of the Cartographic Boundary Files by County dataset on the United States Census Bureau from

  • https://www.census.gov/geographies/mapping-files/time-series/geo/carto-boundary-file.html

    • Unemployment rates data (2000 to 2022) extraced from U.S. Department of Agriculture.

  • https://www.ers.usda.gov/data-products/county-level-data-sets/county-level-data-sets-download-data/

    • Tools used in this project

    • QGIS Version 3.34 to create state and county boundary.
    • 'Generate XYZ Tiles (Directory)' and adding a label with a rule based labeling on the zoom.
    • 'Add Delimited Text Layer' to create unemployment rates layer using csv file.
    • 'Symbology' -> 'Graduated' to create variation of unemployment rates by county and assign color ramp.
    • See this project on my GitHub: Wookienoh09
